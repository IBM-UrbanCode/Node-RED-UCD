This step requires the name of the .json flow file to install and the root directory where NodeRED is installed Install -> Installs NodeRED on the agent host machine. This step requires the root directory where NodeRED is to be installed. To install NodeRED the target machine must have Git and Nodejs installed Start -> Starts NodeRED on the agent host machine. This step requires the root directory where NodeRED is installed and the name of the flow for NodeRED to use. This links with the "Deploy Flow" step Stop -> Stops NodeRED on the agent host machine. This step requires the root directory where NodeRED is installed. Uninstall -> Uninstalls NodeRED from the agent host machine. This step requires the root directory where NodeRED is installed.
